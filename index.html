Week 1 Project: Favorite Animal Website

Make a simple website for your favorite animal.

  • Use the heading, paragraph, image, link, and list HTML tags we covered in Week 1.
  • Add an additional page and link to it from the first page.
  • Use your in-class code file (or this HTML Tags Reference page) as reference.
  • Feel free to use text and image content from the web, but make sure to keep all images and HTML files together in one folder.
