Week 1 Project

Create a 2-page raw HTML website

  • Choose a topic of special interest to you
    • a hobby ?
    • a pet ?
    • an artwork ?
    • a recipe ?
  • Break your content into two separate pages, and use the link (a) tag to link between them
  • Make use of the following HTML tags to markup your content:
    • heading (<h1>, <h2>...)
    • paragraph (<p>)
    • image (<img>)
    • strong and emphasis (<strong> <em>)
    • link (<a>)
    • list (<ul>, <li>)
  • Annotate your code with comments <!-- -->

Tips

  • Start with the base HTML structure (found in class files)
  • Keep all your files (images and HTML files) together in one folder
  • Feel free to use text and image content from the web
  • Keep your focus on:
    • Using HTML tags to add structure & meaning to your content
    • Keeping your HTML code organized
  • Don't worry about formatting/style/making it look pretty

Bonus Challenge

  • Look up the table tag and use it to add a simple table of information to one of your pages
