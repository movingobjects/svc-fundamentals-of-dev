Week 1 Project: Favorite Animal Website

Make a simple website for your favorite animal.

  • Use the heading, paragraph, image, link, and list HTML tags we covered in Week 1.
  • Add an additional page and link to it from the first page.
  • Use your in-class code file (or this HTML Tags Reference page) as reference.
  • Feel free to use text and image content from the web, but make sure to keep all images and HTML files together in one folder.

Futher reading

  • See if you can complete all four challenges!

Week 2 Project: Adding Some Style

Use CSS to style up your Favorite Animal Website.

  • Use the <link> tag to link your CSS file (named style.css) to all pages of your website
  • Use the CSS properties we covered in class to style all the elements of the page, including the <body> element.
  • Upload your site to your own folder on the class website using the provided FTP credentials.
  • EXTRA: Make a copy of your project folder, and try adding a completely unique style to your site, but without altering the HTML in any way

Week 3 Project: Website of Your Choice

Choose a piece of interesting content you'd like to present in a website (simple personal portfolio, showcase of a favorite artist, artwork, restaurant, etc).

  • 3+ simple pages
  • Consistent navigation between pages
  • Use as many of the CSS properties we've covered as possible
  • Use structural HTML elements and IDs & classes as necessary
