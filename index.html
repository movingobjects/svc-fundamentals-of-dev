Week 1 Project

Create a 2-page raw HTML website

  • Choose a topic of special interest to you
    • a hobby ?
    • a pet ?
    • an artwork ?
    • a recipe ?
  • Break your content into two separate pages, and use the link (a) tag to link between them
  • Make use of the following HTML tags to markup your content:
    • heading (<h1>, <h2>...)
    • paragraph (<p>)
    • image (<img>)
    • strong and emphasis (<strong> <em>)
    • link (<a>)
    • list (<ul>, <li>)
  • Annotate your code with comments <!-- -->

Tips

  • Start with the base HTML structure (found in class files)
  • Keep all your files (images and HTML files) together in one folder
  • Feel free to use text and image content from the web
  • Keep your focus on:
    • Using HTML tags to add structure & meaning to your content
    • Keeping your HTML code organized
  • Don't worry about formatting/style/making it look pretty

Bonus Challenge

  • Look up the table tag and use it to add a simple table of information to one of your pages

Week 2 Project

Add CSS to Your Website

  • Create a style.css file
  • Use the <link> tag to link this file to all the pages of your website
  • Use the CSS properties we covered this week to style all of your HTML tags
    • Typography color, font-size, font-style, font-weight, letter-spacing, line-height, text-align, text-decoration, text-transform
    • Spacing margin, padding
    • Backgrounds background-color, background-image, background-repeat, background-size
    • Borders border, border-radius
    • Others opacity, visibility, list-style-type, cursor
  • Use WebFonts from Google Fonts
  • Upload your project to the web following the steps and login information provided in the slides.

Bonus: Two Styles!

  • Make a copy of your complete project folder
  • Clear your style.css file and create a completely unique style from scratch without altering any of the HTML code
